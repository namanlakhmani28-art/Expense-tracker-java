# 💰 Expense Tracker (Java CLI Project)

A simple **Expense Tracker application built using Core Java** that helps users manage income and expenses through a command-line interface.

---

## 🚀 Features

* Add income transactions
* Add expense transactions
* View all transactions
* Calculate current balance
* Organized using OOP principles
* Simple CLI-based interaction

---

## 🧠 Tech Stack

* Java (Core)
* OOP (Classes, Objects)
* ArrayList
* Command Line Interface (CLI)

---

## 📁 Project Structure

```
ExpenseTracker/
│
├── Main.java
├── models/
│   └── Transaction.java
├── services/
│   └── ExpenseService.java
├── .gitignore
```

---

## ▶️ How to Run

### 1. Compile

```bash
javac Main.java models/*.java services/*.java
```

### 2. Run

```bash
java Main
```
